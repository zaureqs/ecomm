# Ecommerce app


## Getting Started

1. clone this repository 
```bash
git clone https://github.com/zaureqs/ecomm
```

2. install dependencies
```bash
pnpm install
```

3. create .env file and set environment variables

4. generate the prisma client
```bash
pnpm db:generate
```

5. start dev server
```base
pnpm dev
```
